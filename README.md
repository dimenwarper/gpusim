# gpusim

Tools for simulating GPUs.

## Overview

`gpusim` is a simulation framework for modeling GPU architectures and workloads, useful for research, education, and performance analysis in science and engineering.

## Features

- GPU architecture simulation (compute units, memory hierarchy, scheduling)
- Workload modeling and profiling
- Performance analysis and visualization tools
